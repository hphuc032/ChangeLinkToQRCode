#  ChangeLinkToQRCODE

Project Python chuyển đổi Link / Text / WiFi / VCard thành mã QR Code  
Hỗ trợ:
- QR cơ bản
- QR WiFi
- QR VCard (Danh thiếp)
- QR tùy chỉnh màu
- QR có logo ở giữa
- Tự động thêm label bên dưới QR

---

##  Tính năng

- Tạo QR từ URL hoặc text
- QR Wi-Fi (quét là kết nối)
- QR danh thiếp (VCard)
- QR có logo (bo tròn + viền trắng)
- Tùy chỉnh màu sắc
- Tự động thêm tên file dưới QR
- Menu CLI dễ sử dụng
